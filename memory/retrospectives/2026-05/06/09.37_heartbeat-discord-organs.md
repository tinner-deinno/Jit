## Lessons Learned

- **Pattern**: Heartbeat adaptive mode via file-drop is a low-friction inter-agent protocol — no sockets, no IPC, just `/tmp` files. Sufficient for this system's tempo.
- **Pattern**: Discord bot memory in a git-tracked JSON file doubles as an audit log — tradeoff is repo size growth over time; consider `memory/discord-memory.json` in `.gitignore` if it grows large.
