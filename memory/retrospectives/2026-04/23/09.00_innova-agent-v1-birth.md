## Lessons Learned

- **Pattern**: Buddhist anatomy = clean separation of concerns สำหรับ AI systems
- **Pattern**: File-based message bus ทำงานได้ดีสำหรับ async multiagent ที่ไม่ต้องการ real-time
- **Pattern**: สร้าง eval scripts (soul-check, body-check) ควบคู่กับ features เสมอ ไม่ใช่ทีหลัง
- **Warning**: ยังไม่มี integration test จริง — ทุก component ดูดีแต่ end-to-end ยังไม่ผ่านมือ
