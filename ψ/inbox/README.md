# ψ/inbox — Jit Inbox

จิต (Jit) รับข้อความจาก:
- innova-bot (ร่างกาย) → via mind-body-bridge skill
- Oracle knowledge sync events
- Agent-to-agent messages

## Format

```json
{
  "from": "innova-bot",
  "subject": "task:implement-feature",
  "body": "...",
  "timestamp": "2026-05-20T10:45:00"
}
```

Place message files here as: `YYYYMMDD_HHMMSS_from-subject.json`
