## Lessons Learned

1. **On Windows, POSIX path encoding in bash differs from expected.** `sed` transformations on `$HOME` paths may not match the Claude project directory key. Always verify the literal project dir name before assuming the encoded form.

2. **Agentic message buses need TTL or read-receipt protocols.** An inbox that fills indefinitely during orchestrator downtime creates operational debt. Design for "what happens when the master is offline for 3 days" — not just the happy path.

3. **Always read the inbox before writing the retro on re-entry.** When re-entering after a gap, Hermes/agent inboxes carry forward-looking signals that change situational awareness. A 5-minute inbox scan before /rrr would make the retro more accurate.
