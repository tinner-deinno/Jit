## Lessons Learned

1. **Broad search before narrow**: เมื่อหา hidden config ใน system ที่ไม่รู้จัก ให้เริ่มจาก broad filesystem grep ก่อน อย่าเริ่มจาก assumption ว่ามันอยู่ที่ไหน

2. **Cloudflare blocks headless HTTP**: API ที่อยู่หลัง Cloudflare ต้องการ User-Agent header ที่ดูเหมือน browser ถ้าไม่มีจะได้ 403 error code 1010 ทันที

3. **Electron apps มี settings อยู่ใน AppData/Roaming**: VS Code-based Electron apps เก็บ user settings ใน `AppData/Roaming/<AppName>/User/settings.json` — ตรวจสอบที่นี่ก่อนแก้ compiled binary
