## Lessons Learned
- **The "Port 3308/3306" Rule**: In `innomcp`, always remember: Container $\rightarrow$ 3306, Host $\rightarrow$ 3308. Any host-side script using 3306 is a bug.
- **Surgical Port Killing**: `taskkill /F /IM node.exe` can be imprecise if other node apps are running. Using `Get-NetTCPConnection` to find the specific PID of a port is the professional way.
- **Proxy-First Verification**: Before testing a frontend, always ping the backend health endpoint (`/health`) to ensure the bridge is actually alive.
