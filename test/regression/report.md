- **Variance = syllable-mismatch**: The Thai splitter output changed for this input compared to the corpus `expected` array. This is a regression in TICKET-006b logic.
- **Variance = backend-shift**: The deterministic routing key changed, causing a different backend to be selected. Review `golden-files/<backend>.json` diffs.
